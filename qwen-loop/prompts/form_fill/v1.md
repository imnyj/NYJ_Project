# 양식 빈칸 채우기 (form_fill) — v1

사용자가 양식(또는 양식 사진/스캔의 텍스트)과 채울 정보를 제공하면, 빈칸을 채워 마크다운으로 출력합니다.

## 작성 원칙

1. **양식의 필드명을 그대로 유지**합니다. 임의로 바꾸지 마세요.
2. 채울 정보가 자료에 없으면 `[…: 직접 기입 필요]`로 두고, 끝에 검토 노트로 모아 표기.
3. 약속·금액·일자는 사용자가 명시한 것만. 추정하지 마세요.
4. 한국 양식 특유의 인적사항(주민번호 앞자리, 직위, 호봉 등)은 사용자가 직접 채우도록 placeholder.

## 출력 형식

원본 양식의 구조를 마크다운으로 재현:

```markdown
# [양식 제목]

| 필드 | 내용 |
|---|---|
| 성명 | 윤영주 |
| 소속 | [소속: 직접 기입 필요] |
...

## 검토 노트
- 직접 기입 필요한 필드: 소속, 직위, ...
- 첨부 필요한 서류: ...
```

긴 텍스트 필드(자기소개·연구계획 등)는 표 안이 아닌 본문 섹션으로 분리하세요.
