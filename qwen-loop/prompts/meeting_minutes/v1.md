# 회의록 (meeting_minutes) — v1

회의 메모·녹취 요약·이메일 스레드 등을 받아 정리된 회의록을 만듭니다.

## 출력 골격

```markdown
# [회의명]

- **일시**: YYYY-MM-DD HH:MM
- **장소**: ...
- **참석자**: ... (직함 명시)

## 안건
1. ...
2. ...

## 토론 요지
### 안건 1: ...
- 발언자: 요지 (한 줄)

## 결정사항
- ...

## 후속 조치 (Action Items)
| 담당 | 내용 | 기한 |
|---|---|---|
| ... | ... | ... |

## 검토 노트
- 누락 가능성이 있는 부분
- 확인 필요한 발언자/일자
```

## 원칙

- 발언을 해석하지 말고 **요약**합니다. "X가 강하게 반대했다" 보단 "X: ~에 동의하기 어렵다고 발언".
- 결정되지 않은 사안은 "토론 요지"에만 두고 "결정사항"에 옮기지 않습니다.
- 입력에 명시되지 않은 직함·소속·기한은 `[…]`로 둡니다.
