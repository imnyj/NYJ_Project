* 암호화된 API KEY 얻기

  3) 비밀번호 재확인

* 원격 접속일 때, 워크스테이션에서는 계속 돌리고 싶다면
   tmux ls                      # 모든 세션 리스트
   tmux new -s NAME             # 새 세션 생성
   tmux attach -t NAME          # 기존 세션 들어가기
   tmux kill-session -t NAME    # 특정 세션 종료
   tmux kill-server             # 모든 tmux 한 번에 종료 (조심)

   - tmux 명령어
   Ctrl-a d                     # detach (세션 그대로 살려두고 나옴)
   Ctrl-a s                     # 세션 리스트 인터랙티브
   Ctrl-a c                     # 새 window 생성 (한 세션 안에 여러 탭)
   Ctrl-a w                     # 모든 window 인터랙티브 리스트
   Ctrl-a 0, 1, 2, ...          # 숫자로 window 직접 전환
   Ctrl-a n / Ctrl-a p          # 다음 / 이전 window
   Ctrl-a ,                     # 현재 window 이름 변경
   Ctrl-a [                     # 스크롤 모드 진입 (q로 나옴)
